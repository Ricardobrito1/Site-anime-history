Nosso site foi criado atraves do meu amor por animes, o anime mudou minha vida, pois me ensinou a ser mais persistente nos meus objetivos, nunca desistir, o treino duro pode supera o talento, tambem me ensinou que nunca você irá conseguir chegar longe/realizar seus grandes sonhos sozinho sempre será nescessario ter pessoas ao seu lado te ajudando, complementando, ensinando nescessario, dado isso decidir criar o site AnimesLife para que outros fãs como eu se identifiquem e se seintam bem navegando no nosso site .
